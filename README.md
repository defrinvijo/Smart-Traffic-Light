OVERVIEW:

The Smart Traffic Light Controller is an Arduino-based intelligent traffic management system designed to improve traffic flow, reduce congestion, and enhance road safety. Unlike conventional traffic lights that operate on fixed timing cycles, this project dynamically adjusts signal timings based on traffic density and includes additional smart features such as pedestrian crossing assistance, emergency vehicle priority, and night mode operation.

This project is suitable for engineering students, embedded systems enthusiasts, and IoT beginners who want to learn about smart city applications using Arduino.



FEATURES:

Traffic Density-Based Signal Control

Uses ultrasonic sensors to detect vehicle density on different roads.
Automatically increases green signal duration for roads with heavier traffic.

Emergency Vehicle Priority

Detects emergency vehicles using an IR sensor.
Immediately grants a green signal to the emergency route.
Helps reduce response time for ambulances and emergency services.

Pedestrian Crossing System

Includes a pedestrian push button.
Stops vehicle traffic and allows pedestrians to cross safely.
Activates a buzzer during crossing time.

Night Mode Operation

Uses an LDR (Light Dependent Resistor) to detect low-light conditions.
Switches traffic signals to blinking yellow mode during nighttime.
Reduces power consumption and improves efficiency.

LCD Countdown Display

Displays remaining signal time on a 16×2 LCD screen.
Provides clear information for drivers and pedestrians.

🔔 Audible Alerts

* Buzzer provides warning sounds during pedestrian crossing operations.
